# 🛒 LocalMart

LocalMart is an AI-powered hyperlocal e-commerce platform that connects local vendors with nearby customers.

## Vision

Empower local businesses with modern e-commerce tools while providing customers with a seamless shopping experience.

## Features

- Customer Marketplace
- Vendor Dashboard
- Admin Panel
- Smart Search
- AI Shopping Assistant
- Online Payments
- Real-time Order Tracking
- Product Recommendations
- Analytics Dashboard

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL

### AI
- Gemini / OpenAI
